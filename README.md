DJD Classic Monokai Theme for Notepad++
=============

Based on the classic Monokai palette by Wimer Hazenberg, this is a Notepad++ port that started from the "One Monokai" XML by mnmlize and has been re-themed to match the original Monokai look-and-feel (not the One-Dark derivative most people call "blue Monokai"). It also adds highlighting for `.yml` / `.yaml` files out of the box.

Tested Languages
----------
PHP / CSS / HTML / JavaScript / XML / YAML / Ruby / C# / C++ / SQL / Perl / Pascal / Python

Installation
--------------------------

1. Download [`DJD_Classic_Monokai.xml`](./DJD_Classic_Monokai.xml)
2. Open Notepad++ -> Settings -> Import -> Import style theme
 - OR
2. Start -> Run -> type `%APPDATA%/Notepad++`
3. Copy this file to `themes` folder then Restart Notepad++.
4. Open `Settings->Style Configurator`.
5. Select `DJD Classic Monokai` from the theme drop-down box.
6. Click *Save & Close*
